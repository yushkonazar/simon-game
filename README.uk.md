Браузерна реалізація класичної гри **Simon Says**. Спостерігай за послідовністю кольорів, що засвічуються — і повтори її. З кожним раундом додається ще один крок. Як далеко ти зайдеш?

## 🔗 Демо

👉 [Відкрити на GitHub Pages](https://yushkonazar.github.io/simon-game/)

## ✨ Функціональність

- Класичний геймплей Simon Says з наростаючою складністю
- Підтримка клавіатури: клавіші **W** / **A** / **S** / **D**
- Підтримка кліків мишею
- Унікальний звуковий ефект для кожної кнопки + сигнал програшу
- Візуальне підсвічування кнопок при активації
- Ретро-аркадна стилістика

## 🛠 Технології

- HTML5
- CSS3 (анімації, ретро-стиль)
- JavaScript (Vanilla JS, без бібліотек)
- MP3 звукові файли

## 🚀 Запуск

**Варіант 1 — GitHub Pages (без налаштувань):**

👉 [https://yushkonazar.github.io/simon-game/](https://yushkonazar.github.io/simon-game/)

**Варіант 2 — Локальний запуск:**

1. Клонуй репозиторій:
   ```bash
   git clone https://github.com/yushkonazar/simon-game.git
   cd simon-game
   ```
2. Відкрий `index.html` напряму у браузері — сервер та залежності не потрібні.

## 🎮 Як грати

1. Натисни будь-яку клавішу (або клікни кнопку), щоб почати гру
2. Спостерігай за послідовністю кольорів, що засвічуються
3. Повтори послідовність за допомогою **клавіатури** (W/A/S/D) або **кліків мишею**:

   | Клавіша | Колір    |
   |---------|----------|
   | W       | Зелений  |
   | A       | Червоний |
   | S       | Жовтий   |
   | D       | Синій    |

4. Кожен успішний раунд додає ще один колір до послідовності
5. Зробив помилку — гра закінчена! Натисни будь-яку клавішу, щоб перезапустити
