A browser-based implementation of the classic **Simon Says** memory game. Watch the sequence of colored buttons light up, then repeat it — each round adds one more step. How far can you go?

## 🔗 Live Demo

👉 [Open on GitHub Pages](https://yushkonazar.github.io/simon-game/)

## ✨ Features

- Classic Simon Says gameplay with progressive difficulty
- Keyboard support: **W** / **A** / **S** / **D** keys
- Mouse click support
- Unique sound effect for each button + game-over buzzer
- Visual button glow on activation
- Retro arcade aesthetic

## 🛠 Tech Stack

- HTML5
- CSS3 (animations, retro styling)
- JavaScript (Vanilla JS, no libraries)
- MP3 sound files

## 🚀 Getting Started

**Option 1 — GitHub Pages (no setup needed):**

👉 [https://yushkonazar.github.io/simon-game/](https://yushkonazar.github.io/simon-game/)

**Option 2 — Run locally:**

1. Clone the repository:
   ```bash
   git clone https://github.com/yushkonazar/simon-game.git
   cd simon-game
   ```
2. Open `index.html` directly in your browser — no server or dependencies needed.

## 🎮 How to Play

1. Press any key (or click a button) to start the game
2. Watch the sequence of colors that lights up
3. Repeat the sequence using **keyboard** (W/A/S/D) or **mouse clicks**:

   | Key | Color  |
   |-----|--------|
   | W   | Green  |
   | A   | Red    |
   | S   | Yellow |
   | D   | Blue   |

4. Each successful round adds one more color to the sequence
5. Made a mistake — game over! Press any key to restart
